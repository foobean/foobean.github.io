hexo3.0下配制git问题

发表于 2015-08-18   |   分类于 技术   |  

hexo配制网上比较多，下面是我配制3.0版本遇到问题

git配制

官方https://hexo.io/docs/deployment.html#Git

deploy: type: git(后面是有个空格的,hexo3.0之后type填写git)

repo: git@github.com:username/username.github.io.git

部署

hexo generate
hexo deploy #配好git后会把public目录下静态文件部署到git上
或
hexo d -g

常用命令：
hexo clean
hexo d -g #生成加部署
hexo s -g #预览加部署
简写：
hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy

现在可以访问你的地址了 http://username.github.io/

no layout: index.html

这个问题因该是hexo下_config.yml配制有问题，最好仔细检查一下或重新初始化

参考
http://www.zhihu.com/topic/19851557/newest
